aSimpleMU v0.6.9b Help System

Topics:

commands - list of all the available commands and their permission requirements
movement - how moving around works in this game
olc      - the online creation system
help     - this page
license  - the license for this software
policies - in-game, and out-of-game policy information

'help <topic/command>' Show the help for the given topic/command.
'help @reload' Reload the help files while the game is running.
               Suitable for updating help after manually editing a help file
               without a shutdown and restart.
               
* topic names, etc are case sensitive and are usually lowercase
